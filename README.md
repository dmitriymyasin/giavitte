# Портал "Корочки.есть"

Информационная система для записи на онлайн курсы дополнительного профессионального образования.

## Требования

- Python 3.8 или выше
- MySQL 5.7 или выше (Создать из docker образа базу данных MySQL (Doker_create_db_mysql))
- Git (для скачивания проекта)

## Установка и запуск

#### Скачивание bootstrap
